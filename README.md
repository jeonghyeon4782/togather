# "일정관리부터 정산까지 TOGATHER!!"

## 어떤 사람들을 위한 서비스일까요?
- 각자 가고 싶은 장소를 한 번에 볼 수 없어서 불편해요.
- 일일이 계산하지 않고 한 번에 정산하고 싶어요.
- 개인별 정산을 할 수 있으면 좋겠어요.
